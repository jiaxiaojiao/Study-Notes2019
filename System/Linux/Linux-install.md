### 安装vim

    yum -y install vim*

### 安装rz,sz命令

```text
yum install lrzsz
```

上传文件 `rz filename`

下载文件 `sz filename`


### 安装autoconf库
```text
yum -y install autoconf
```

### 安装wget
```text
yum -y install wget
```

### 安装[zip](Linux-zip.md)和[unzip](Linux-unzip.md)

    apt-get install zip
    apt-get install unzip
    # 或者
    yum install zip
    yum install unzip
